sendAndDecode( ) : \mixed

Convenience method for sending a request and decoding the response

This calls the {@see send()} method and then the {@see Message::getJsonDecodedBody()} method, and returns the result.

This function accepts parameters which will be passed directly to {@see Message::getJsonDecodedBody()}. This can be generally used to pass extra arguments to json_decode.
